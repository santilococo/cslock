## Requirements <a name="requirements"></a>

In order to build `slock` you need the `Xlib` header files.

## Installation <a name="installation"></a>

You need to run:

```bash
make clean install  
```

## Usage <a name="usage"></a>

To run `cslock`:

```bash
slock
```

Then, to get out of it, enter your password.
